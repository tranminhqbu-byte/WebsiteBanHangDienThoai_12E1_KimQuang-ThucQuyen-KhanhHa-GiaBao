# DiDongVui Website

## Giới thiệu

Dự án website bán điện thoại (DiDongVui) được xây dựng bằng HTML và CSS. Website mô phỏng một hệ thống bán hàng cơ bản với giao diện hiện đại, responsive và đầy đủ các trang chức năng.

## Tính năng chính

* Trang chủ (Home)
* Trang sản phẩm (Shop)
* Trang chi tiết sản phẩm (Product Details)
* Trang blog/tin tức
* Trang giới thiệu (About)
* Trang liên hệ (Contact)
* Trang giỏ hàng (Cart)
* Responsive cho nhiều thiết bị
* Hiệu ứng hover và UI đồng bộ

## Công nghệ sử dụng

* HTML5
* CSS3 (External CSS)
* Font Awesome (icon)

## Cách chạy dự án

* Mở Link Live đã deploy lên github 
* Sử dụng Live Server trong VS Code
